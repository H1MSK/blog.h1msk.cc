# 简介

(以下内容译自官网)
Typora 既是一个阅读器，同时也是一个在线编辑器。
它取消了 Markdown 源码的预览窗口、模式选择器、语法符号，以及所有其他不必要的干扰。
反之，它提供了一个真正的在线同步预览功能来让你专注于内容本身。
