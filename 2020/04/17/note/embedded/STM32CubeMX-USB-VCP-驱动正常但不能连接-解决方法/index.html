# 软硬件版本

  • Windows 1909 或 2004 Preview
  • STM32F407VET6
  • STM32CubeMX 5.4.0 / 5.6.0 / 5.6.1
  • USB 库 v1.0_Cube

# 症状描述

  • Windows 可检测到虚拟串口，无感叹号（时序设置正确）
  • Windows API 中，
    • CreateFileA 正常
    • SetCommState 失败， GetLastError 返回 56
  • PuTTY 无法连接串口，TeraTerm 正常
